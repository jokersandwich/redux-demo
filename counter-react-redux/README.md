# a counter with react, redux and react-redux

<strong>笔记</strong>

redux项目常用json依赖：

（版本用最新的就行）

    "dependencies": {

      //babel
      
      "babel-core": "^6.4.5",

      "babel-loader": "^6.2.1",

      "babel-plugin-transform-es2015-arrow-functions": "^6.4.0",

      "babel-preset-es2015": "^6.3.13",

      "babel-preset-react": "^6.3.13",

      //react
      
      "react": "^0.14.6",

      "react-dom": "^0.14.6",

      "react-redux": "^4.0.6",

      "redux": "^3.0.5",

    }

`npm init` 创建package.json文件，大致添加上面这些依赖，redux就可以运行。
